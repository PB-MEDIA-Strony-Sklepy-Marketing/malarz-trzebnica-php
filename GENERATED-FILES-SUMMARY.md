## Quick Start

### 1. Instalacja Lokalna

```bash
# Clone repository
git clone https://github.com/PB-MEDIA-Strony-Sklepy-Marketing/malarz-trzebnica-php.git
cd malarz-trzebnica-php

# Install PHP dependencies
composer install

# Install Node dependencies
npm install

# Copy environment file
cp .env.example .env

# Edit .env with your credentials
nano .env

# Start local server
php -S localhost:8000 -t dist/
```

### 2. Docker Development

```bash
# Start containers
docker-compose up -d

# Access application
open http://localhost:8000
```

### 3. Development Workflow

```bash
# Create feature branch
git checkout -b feature/my-feature

# Make changes...

# Lint code
composer run lint

# Run tests
composer test

# Commit with conventional commits
git commit -m "feat(contact): Add spam protection"

# Push and create PR
git push origin feature/my-feature
```

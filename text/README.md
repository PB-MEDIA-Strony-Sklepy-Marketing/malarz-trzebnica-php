# Przewodnik README - text/

Ten katalog zawiera pliki tekstowe z treściami dla każdej podstrony serwisu Malarz Trzebnica.

## Struktura Plików

| Plik | Strona | Opis |
|------|--------|------|
| `homepage.md` | `/` | Treści dla strony głównej |
| `offer.md` | `/oferta` | Opis usług malarskich |
| `gallery.md` | `/galeria` | Opisy zdjęć w galerii |
| `contact.md` | `/kontakt` | Dane kontaktowe, FAQ |

## Format Treści

Każdy plik używa Markdown z front matter:

```markdown
---
title: "Tytuł strony"
description: "Meta description dla SEO"
keywords: "słowa, kluczowe, seo"
---

# Główny nagłówek

Treść strony w Markdown...
```

## Edycja Treści

Aby edytować treści:

1. Otwórz odpowiedni plik `.md`
2. Edytuj tekst
3. Commit i push zmian
4. GitHub Actions automatycznie wdroży zmiany

## Wskazówki

- **H1** - tylko jeden na stronie (główny tytuł)
- **H2-H4** - struktura hierarchiczna
- **Keywords** - naturalne, nie spam
- **CTA** - jasne wezwania do działania
- **Długość** - min. 300 słów dla SEO
